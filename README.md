# Detection-of-COVID-19-from-Cough-Sounds
• Developed and optimized machine learning and deep learning models
(BLSTM, LSTM, CRNN, RF, CBAM) for COVID-19 detection from
cough recordings, achieving high accuracy in a large-scale dataset (36 GB) of anonymized patient audio samples
•This project explores the use of machine learning and deep learning models, such as SVM, Random Forests, and CNNs (ResNet18, ResNet50, ResNet101), to classify audio data across diverse domains, including environmental sound recognition (ESC-10, ESC-50) and health diagnostics 
•Utilizes multiple datasets, including ESC-10 , ESC-50 , and COUGHVID , emphasizing dataset-specific challenges, evaluation strategies (5-fold cross-validation), and detailed performance metrics (accuracy, precision, recall, F1-score)
•Focuses on processing audio as time-series and spectral data, addressing challenges like noise, variability, and preprocessing requirements, while employing advanced techniques like spectrogram analysis and feature engineering for robust and efficient classification.
• Implemented advanced audio feature extraction techniques(MFCC, ZCR) and converted features into spectrogram images
• Conducted comprehensive data preprocessing, visualization (t-SNE, UMAP), and rigorous model evaluation, contributing valuable insights for audio-based medical diagnostics
